# Flask Calculator App

A simple and modern calculator web application built using **Python Flask**, **HTML**, **CSS**, and **JavaScript**.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Responsive UI
* Flask backend
* Fast calculations
* Clean dark-themed interface

---

# Tech Stack

* Python
* Flask
* HTML5
* CSS3
* JavaScript

# Installation

## 1. Clone the Repository

```bash
git clone <your-github-repo-link>
```

## 2. Move into Project Folder

```bash
cd calculator
```

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## 4. Run the Flask App

```bash
python app.py
```
